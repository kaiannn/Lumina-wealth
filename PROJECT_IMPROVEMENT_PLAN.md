# Lumina Wealth 改进计划

## 目标

围绕当前项目在财经口径、数据展示、分析准确性上的问题，分阶段完成从“可视化原型”到“具备基础财经严谨性的数据分析产品”的升级。

## 六、可视化体系升级：从“展示”到“洞察”

基于认知科学和用户体验原则，重构可视化体系，让图表不仅是数据展示，更是认知工具和决策辅助。

### 17. 可视化认知价值提升
- [ ] **降低认知负荷**：复杂数据 → 简单图形，抽象概念 → 具体形象
- [ ] **加速决策过程**：提供决策支持可视化，对比不同策略效果
- [ ] **行为反馈机制**：即时看到行为后果，识别行为模式
- [ ] **习惯养成工具**：建立数据驱动的决策习惯，培养长期视角

### 18. 结构化可视化体系
- [ ] **持仓结构可视化**：树状图展示资产层级，识别过度集中
- [ ] **资产配置环形图**：sunburst图展示资产类别分布平衡性
- [ ] **成本-价格对比图**：ECharts折线图对比持仓成本与市场价格
- [ ] **交易行为时间轴**：自定义组件展示交易历史和行为模式
- [ ] **风险-收益散点图**：评估每项投资的风险收益特征
- [ ] **相关性矩阵热力图**：展示资产间相关性，评估分散化效果
- [ ] **金字塔结构图**：堆叠柱状图展示持仓成本结构（正/倒金字塔）
- [ ] **行为模式雷达图**：综合评估投资行为特征和改进方向

### 19. 时间维度深度洞察
- [ ] **持仓演变时间轴**：动态展示持仓结构历史变化
- [ ] **行为模式时间序列**：分析交易频率、仓位调整节奏
- [ ] **成本分布可视化**：识别成本集中区域和不合理买入时机
- [ ] **行业集中度热力图**：可视化行业风险暴露

### 20. 智能可视化功能
- [ ] **智能洞察生成**：AI驱动自动发现模式、异常、机会
- [ ] **个性化适配**：根据用户类型（新手/中级/高级）调整图表复杂度
- [ ] **交互式探索**：支持下钻分析、时间滑动、假设分析
- [ ] **多设备适配**：响应式设计，优化移动端触摸体验
- [ ] **故事化叙述**：每张图表讲述完整故事，引导用户发现洞察

### 21. 可视化效果评估体系
- [ ] **用户理解度指标**：测量用户对数据的理解程度
- [ ] **决策质量指标**：评估基于可视化的决策效果
- [ ] **使用频率监控**：跟踪可视化工具的使用频率
- [ ] **技术性能指标**：监控渲染速度、交互响应、准确性
- [ ] **可访问性保障**：考虑色盲用户需求，提供完整上下文

## 十、成功标准

当以下条件满足时，可以认为这个项目从"展示型原型"进入"基础财经可用版本"：
- 用户不会被错误命名误导
- 收益可以拆分为已实现和未实现
- 汇总口径在单一基准币种下统一
- 时间轴展示的是正确的累计逻辑
- 对比模块不再仅停留在"同名股票重合"
- 页面上能清楚说明每个指标的计算口径

### 可视化体系成功标准
- 用户能够通过可视化发现隐藏的模式和风险
- 可视化加速用户从数据到洞察的认知过程
- 图表不仅是数据展示，更是行为改变工具
- 个性化适配不同用户类型的认知需求
- 可视化工具成为用户日常投资决策的核心辅助
